Любой сценарий PHP состоит из последовательности инструкций. Инструкцией может быть присваивание, вызов функции, повтор кода (цикл), сравнение, или даже инструкция, которая ничего не делает (пустой оператор).

После инструкции обычно ставится точка с запятой.

Список управляющих конструкций

  • if
  • else
  • elseif/else if
  • Альтернативный синтаксис управляющих структур
  • while
  • do-while
  • for
  • foreach
  • break
  • continue
  • switch

Список управляющих конструкций (продожение)

  1. declare
  2. return
  3. require
  4. include
  5. require_once
  6. include_once
  7. goto
